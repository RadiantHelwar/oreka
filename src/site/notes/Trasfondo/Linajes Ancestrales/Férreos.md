# Férreos

Descendientes de lo Geran el Indomable

## Dote de Origen

### Férreo

**Requisitos**: Orco o con ascendencia Orca.

Tu cuerpo porta una esencia metálica latente, que le permite endurecerse o dispersarse en momentos de peligro.

Ganas los siguientes beneficios.

**Resistencia Metálica.** Puedes usar este rasgo un número de veces igual a tu Bonificación por Competencia, y recuperas todos los usos gastados cuando terminas un descanso largo.

Puedes usar este rasgo de una de las siguientes formas:

- **Endurecer.** Cuando recibes daño, puedes usar tu reacción para ganar resistencia a ese daño.
- **Dispersar.** Cuando realizas una tirada de salvación de Fuerza, Destreza o Constitución, puedes obtener ventaja en esa tirada.
